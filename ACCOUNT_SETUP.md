# MatMind Account Setup

Use this to complete mandatory publisher accounts.

## Apple Developer (required for iOS)

1. Enroll at https://developer.apple.com/programs/
2. Complete identity verification and payment.
3. Open App Store Connect and create the app record:
   - Bundle ID: `com.matmind.app`
   - Platform: iOS
4. Invite team members with release permissions.

## Google Play Console (required for Android store)

1. Register at https://play.google.com/console/
2. Complete organization and identity verification.
3. Create app record:
   - Package name: `com.matmind.app`
   - Default language and app category
4. Configure testers for Internal and Closed tracks.
